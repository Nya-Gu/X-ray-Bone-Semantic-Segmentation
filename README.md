# X-ray Bone Semantic Segmentation

## 0. 프로젝트 소개 및 목차
**(1) 개요**: 손을 촬영한 X-ray 이미지에서 29개의 손가락, 손목, 팔뼈를 세그멘테이션하는 모델을 개발하는 프로젝트입니다.

**(2) 목표**: 모델 성능 평가 지표는 class mean dice로, 29개의 클래스의 dice를 평균 낸 값을 높이는 것이 이번 프로젝트의 목표입니다.

**(5) 유의사항**
dice는 0~1 사이의 값이지만 소수점을 그대로 표기할 경우 (그리고 소수점이 길어질수록) 읽기 불편한 면이 있는 것 같습니다. 그래서 편의상 100을 곱해 % 단위로 표기하기로 했습니다.

예시: dice=0.9903 -> 99.03%
